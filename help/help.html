Welcome to the Visual Flow Help Page!

ERRORS
MODEL Line Die MODEL Zeile ist obligatorisch. In ihr können Sie dem Modell einen Namen geben.
Syntax: MODEL="text" wobei text alle Zeichen ausser " enthalten darf. text darf auch leer sein; dies wird aber nicht empfohlen.
VERSION Line Die VERSION Zeile ist obligatorisch. In ihr können Sie dem Modell einen Version geben.
Syntax: VERSION="text" wobei text alle Zeichen ausser " enthalten darf. text darf auch leer sein; dies wird aber nicht empfohlen.
OUTPUT Line Die OUTPUT Zeile ist obligatorisch. In ihr können Sie den Namen einer Datei bestimmen, in welcher das Resultat gespeichert wird.
Syntax: OUTPUT="text" wobei der Dateiname *.xls heissen sollte.
TYPE Line Die TYPE Zeile ist obligatorisch. In ihr bestimmen sie den Typ der Simulation. Es stehen Constant und Normal zur Verfügung. Bei der Auswahl von Normal werden die Werte als Wahrscheinlichkeitsverteilung dargestellt. Das Setzen einer Varianz (VAR) ist erforderlich.
Syntax: TYPE=Constant, TYPE=Normal VAR=x wobei x eine positive reelle Zahl ist
RUNS Line Die RUNS Zeile ist obligatorisch. In ihr bestimmen sie die Anzahl der Simulation. Es wird eine Ganzzahl grösser 0 verlangt.
Syntax: RUNS=x wobei x eine positive ganze Zahl ist
STEPS Line Die STEPS Zeile ist obligatorisch. In ihr bestimmen sie die Anzahl Iterationen in einer Simulation. Es wird eine Ganzzahl grösser 0 verlangt.
Syntax: STEPS=x wobei x eine positive ganze Zahl ist
PRECISION Line Die PRECISION Zeile ist obligatorisch. In ihr bestimmen sie die Präzision der Berechnungen. Es wird eine Potenz von 10 kleiner 0 verlangt.
Syntax: PRECISION=x wobei x zum Beispiel 0.1, 0.01 usw. ist
COMPARTMENT Line Ein Modell besteht aus einem oder mehreren COMPARTMENT Zeilen.
Syntax: COMPARTMENT a=x wobei a ein eindeutiger Name ist und x eine Zahl
PARAMETER Line Ein Modell besteht aus einem oder mehreren PARAMETER Zeilen.
Syntax: PARAMETER a=x oder PARAMETER a=x STD=y (für TYPE=Normal) wobei a ein eindeutiger Name ist, x eine Zahl und y >0
DELTA Line Ein Modell besteht aus einem oder mehreren DELTA Zeilen. Sie dient dazu, die Veränderung pro Zeiteinheit für ein Compartment anzugeben. Es handelt sich um einen mathematischen Ausdruck.
Syntax: DELTA a=x oder DELTA a=x STD=y (für TYPE=Normal) wobei a ein eindeutiger Name ist, x eine Zahl und y >0
